# Blog
plicação criada para simular um Blog. A funcionalidade do blog é a criação de categorias e artigos, somente usuarios registrados podem criar categorias ou artigos, para manter a sessão do usuario ativa foi utilizado a biblioteca express-session. Foi implementado TinyMce como editor de texto.
## Front-End
EJS como view engine, bootstrap para estilização da pagina.
## Back-End
- Node.Js, Express como framework.
- ORM do projeto foi utilizado o sequelize.
- Bcrypt para fazer o Hash da senha, para cadastrar o usuario.
